本文介绍如何使用 Grails 快速创建一个 Web 应用，使用 Groovy 语言编写一个简单的 “Hello World”，使用 Gradle 构建并运行起来。

Grails 是一个全栈式框架，基于 Spring / Spring Boot 和 Hibernate 技术之上构建，通过其核心技术如 GORM、GSP、JSON 视图、插件，借助 Groovy 动态语言和强大的 DSL 使得构建现代 Web 应用更加容易。

Grails 从 3.0 开始基于 Spring Boot 构建，因此完全可以看做是一个 Spring Boot 框架，在其基础上借助 Groovy 语言的动态语言特性和强大的 DSL 表达能力，学习起来非常容易。目前 Grails 最新版本是 3.3.9，基于 Spring Boot 版本是 1.5.18，4.0 将会升级 Spring Boot 到 2.1，而底层使用 Micronaut for Spring 重写，与 Micronaut 优势完美结合在一起。

1. 介绍

内容简介： 一个简单的 Grails Web 应用示例
语言框架： Groovy、Grails
难度级别： L1
阅读时间： 10 分钟

2. 工具准备

  • Java SDK : 8.0

  • Grails : 3.3.9

  • Gradle : 4.10.2

  • IntelliJ IDEA : 2018.3

  • Curl / HTTPie

3. 实现步骤

3.1. 创建项目

首先，通过 Grails CLI 创建一个空白工程： Hello World 应用。

使用 Spring CLI 创建一个新工程
$ grails create-app org.springdev.guides.helloworld \
--profile rest-api --inplace

3.2. 打开项目

打开 IDEA，点击菜单 File  Open，选择项目所在目录 grails-hello-world 下的 build.gradle，打开。

3.4. 编写代码

3.4.1. 编写 Controller

接下来我们开始创建一个 Controller。

使用命令： grails create-controller，如下：

$ grails create-controller HelloController

默认生成的

src/main/groovy/org/springdev/guides/helloworld/HelloController.groovy
package org.springdev.guides


import grails.rest.*
import grails.converters.*

class HelloController {  (1)
  static responseFormats = ['json', 'xml']  (2)

    def index() {  (3)
        render "Hello, World!"
    }
}
代码说明如下：
1 Grails 约定后缀包含 Controller 就可以，这是一个 Grails 的 REST 接口类；
2 设置响应的内容格式，默认支持 json 和 xml，在这个例子中可以不需要，因为我们返回的是 text/html；
3 响应成功，并返回文本内容为：Hello, World!。

3.5. 单元测试

3.5.1. 编写测试用例

默认在创建 Controller 的同时，脚手架已经也创建了单元测试类 HelloControllerSpec.groovy，这是一个 Spock 测试类。

src/test/groovy/org/springdev/guides/HelloControllerSpec.groovy
package org.springdev.guides

import grails.testing.web.controllers.ControllerUnitTest
import spock.lang.Specification

class HelloControllerSpec extends Specification
  implements ControllerUnitTest<HelloController> {  (1)

    def setup() {
    }

    def cleanup() {
    }

    void "Testing Hello Web API"() {  (2)
        when:"The default index action is invoked"
        controller.index()  (3)

        then:"Hello World"
        response.text == "Hello, World!"  (4)
    }
}
代码说明如下：
1 HelloControllerSpec 实现了 ControllerUnitTest，通过利用 grails-test 提供的能力，编写单元测试非常简单；
2 编写一个单元测试用例；
3 调用 HelloController.index()；
4 验证返回内容为文本：Hello, World!。

3.5.2. 执行单元测试

$ grails test-app -unit
$ open build/reports/tests/test/index.html

报告显示测试执行成功，说明功能实现正确。

4. 构建运行

你可以先打包成可执行的 JAR：

$ grails package

或者，

$ gradle build

接着在控制台运行：

$ java -jar build/libs/grails-hello-world-0.1.war

或者，直接执行命令 grails run-app 或者 gradle bootRun:

$ grails run-app
控制台运行日志
| Running application...
Grails application running at http://localhost:8080 in environment: development
