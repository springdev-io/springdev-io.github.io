项目一览

从配置到安全，从Web应用到大数据——无论您的应用需要任何类型的基础设施，这里都有Spring项目帮助您来构建它。选您所选，用您所用——Spring正是由模块化而设计。

Spring Boot

构建所想要的Spring应用程序，并且快速启动和运行起来。
Spring Framework

为依赖注入，事务管理，Web应用程序，数据访问，消息传递等提供核心支持。
Spring Cloud Data Flow

用于可组合数据的现代运行时微服务应用的编排服务。
Spring Cloud

提供一组用于构建和部署微服务等分布式系统中常见模式的工具。
Spring Data

提供一致的数据访问方法 —— 关系，非关系，大数据处理等。
Spring Integration

通过轻量级消息传递和声明式适配器支持众所周知的企业集成模式。
Spring Batch

简化并优化大量批处理操作的工作。
Spring Security

通过全面和可扩展的身份验证和授权支持来保护您的应用程序。
Spring HATEOAS

简化创建遵循HATEOAS原则的REST风格服务。
Spring REST Docs

通过将手写文档与使用Spring MVC Test或REST Assured自动生成的片段相结合，来记录RESTful服务。
Spring AMQP

将核心Spring概念应用于基于AMQP的消息传递解决方案的开发。
Spring Mobile

通过设备检测和渐进式渲染选项简化移动Web应用程序的开发。
Spring for Android

提供用于开发Android应用程序的关键Spring组件。
Spring Web Flow

支持使用受控导航构建Web应用程序，例如航班登机或贷款申请。
Spring Web Services

便于契约优先SOAP Web服务的开发。
Spring LDAP

使用Spring熟悉的基于模板的方法简化使用LDAP的应用程序开发。
Spring Session

Spring Session提供了用于管理用户会话信息的API和实现。
Spring Shell

为使用基于Spring的编程模型构建命令行应用程序提供了强大的基础。
Spring Flo

一个JavaScript库，为管道和简单图形提供基本的可嵌入HTML5可视化构建器。
Spring Kafka

为Apache Kafka提供熟悉的Spring抽象。
Spring Statemachine

为应用程序开发人员提供可在Spring应用程序中使用状态机概念的框架。
Spring IO Platform

为构建现代应用程序提供了一个内聚的版本化平台。它是一种模块化的企业级分发版，可提供一组精心选择的依赖项。

Community Projects

Spring Roo

使用它可以在几分钟内快速轻松地构建完整的Java应用程序。
Spring Scala

将Scala的强大功能和表现力与Spring的生产力和深层生态系统结合在一起。

Projects in the Attic

Spring BlazeDS Integration

为使用Adobe Flex前端客户端的基于Spring的应用程序中的Adobe BlazeDS提供一流的支持。
Spring Loaded

通过在应用程序的JVM中重新加载类文件更改来提高开发效率。
REST Shell

通过基于CLI的资源发现和交互，可以更轻松地编写和测试RESTful应用程序。
Spring XD

通过解决摄取，分析，批处理作业和数据导出，简化大数据应用程序的开发。
Spring Social

轻松将您的应用程序与第三方API（如Facebook，Twitter，LinkedIn等）连接起来。
