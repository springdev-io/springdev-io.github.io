第一个里程碑建立在Spring Framework 5.1.RC1、Spring Data Lovelace RC1和Spring Security 5.1 M2等最新的项目之上。

  • 采用Undertow 2.0 and Tomcat 9.0，支持Servlet 4

  • 缓存端点

  • 支持查询单个组件的运行状况

  • 支持多配置文件YAML文档中的配置文件表达式和<springProfile> Logback配置

Spring Boot 2.0.4 现已可用

release 
rain
2018-07-30
评论
Spring Boot 2.0.4 现已可用，包含了90个修复、改进和依赖项更新。

Spring Boot 2.0.4 包含了90个修复、改进和依赖项更新。

其中修复的部分BUG

  • 当使用EndpointRequest.toAnyEndpoint()时/actuator/不安全

  • 提供一致的方法来发现主要DispatcherServlet的路径

  • 项目特定设置未在Eclipse中应用

  • 由于缺少默认构造函数而无法绑定的属性很难诊断

  • 在使用JdbcTemplate之前，数据库迁移可能尚未运行

  • RedisCacheConfiguration缺少@ConditionalOnClass检查
