release 
rain
2019-01-22
评论
Spring Cloud Task 2.1.0 GA 发布，支持Spring Boot 2.1.0。

Spring Cloud Task 允许用户使用 Spring Cloud 开发和运行短暂的微服务，并在本地运行或云上，甚至在 Spring Cloud Data Flow 上运行。只需在 Spring Boot 应用程序（单个应用程序上下文）中添加 @EnableTask 并可运行。

Spring Cloud Task 2.1.0.RELEASE 现已发布，可以在 Github 和 Pivotal 下载库中找到。
