概要
本文介绍如何快速创建一个Spring Boot Web应用，然后编写一个示例页面并运行它。

创建一个新工程

在经过前面几个章节的准备，我们已经搭建好了本地开发环境，是时候开始热身准备创建一个Spring应用了。接下来我们要通过一步一步的介绍如何从创建空白工程，然后简单写一个Hello World应用，是的，它非常简单，但是足够可以说明我们已经开始上路了，这是万里长征的第一步。

使用Spring CLI 创建一个新工程 
$ mkdir hello-world
$ cd hello-world
$ spring init --name hello-world --artifactId hello-world --groupId com.rainboyan.securitybook --language java --boot-version 1.5.13.RELEASE --type maven-project --dependencies web,thymeleaf,devtools --extract

导入刚才创建的工程

打开STS，点击菜单File  Import，选择导入向导Maven下的Existing Maven Projects，在下一步中选择刚刚创建的工程目录，确定后导入到STS中。

代码一览

此时创建的新工程hello-world目录结构如下:

├── mvnw
├── mvnw.cmd
├── pom.xml
└── src
    ├── main
    │   ├── java
    │   │   └── com
    │   │       └── rainboyan
    │   │           └── securitybook
    │   │               └── helloworld
    │   │                   └── HelloWorldApplication.java
    │   └── resources
    │       ├── application.properties
    │       ├── static
    │       └── templates
    └── test
        └── java
            └── com
                └── rainboyan
                    └── securitybook
                        └── helloworld
                            └── HelloWorldApplicationTests.java

在Package Explorer中打开应用主程序文件：HelloWorldApplication.java，可以看到这个文件代码非常简单。

src/main/java/com/rainboyan/scuritybook/helloworld/HelloWorldApplication.java 
package com.rainboyan.securitybook.helloworld;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class HelloWorldApplication {

	public static void main(String[] args) {
		SpringApplication.run(HelloWorldApplication.class, args);
	}
}

$ mvn spring-boot:run

待启动完成后，打开浏览器，输入地址：http://localhost:8080/，将会看到下面的空白错误页面：

不用奇怪，因为到目前为止，我们并没有开始创建任何页面。

开始编写代码了

接下来我们开始编写欢迎页面。在src/main/resources/templates目录中新增一个index.html，内容如下：

src/main/resources/templates/index.html 
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" xmlns:th="http://www.thymeleaf.org">
    <head>
        <title>Spring Security Book</title>
    </head>
    <body>
        <h1>Hello, World!</h1>

    </body>
</html>

编写MvcConfig.java类，设置首页视图。

src/main/java/com/rainboyan/securitybook/helloworld/MvcConfig.java 
package com.rainboyan.securitybook.helloworld;

import org.springframework.context.annotation.Configuration;
import org.springframework.web.servlet.config.annotation.ViewControllerRegistry;
import org.springframework.web.servlet.config.annotation.WebMvcConfigurerAdapter;

@Configuration
public class MvcConfig extends WebMvcConfigurerAdapter {

    public void addViewControllers(ViewControllerRegistry registry) {
        registry.addViewController("/").setViewName("index");
    }

}
