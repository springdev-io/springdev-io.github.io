Spring Vault 2.1.1、2.0.3 和 1.1.3 发布

release 
rain
2018-11-14
评论

Spring Vault提供熟悉的Spring抽象和客户端支持，用于访问、存储和撤销机密。它提供了与Vault交互的低级和高级抽象，使用户免于底层接口问题。

通过HashiCorp的Vault，你可以在所有环境中为应用程序管理外部机密数据。Vault可以管理静态和动态机密，例如应用程序数据，远程应用程序/资源的用户名/密码，并为外部服务提供凭据，如MySQL、PostgreSQL、Apache Cassandra、Consul、AWS等。

Spring Vault 2.1.1、2.0.3 和 1.1.3 已发布，可以在 Maven Central 下载。

这些版本带来了错误修复和依赖版本更新。Spring Cloud Vault 将在接下来的几天内通过 Spring Cloud Greenwich M2 发布 Spring Vault 2.1.1。

有关更改的完整列表，请参阅更改日志：

  • Spring Vault 2.1.1 RELEASE

  • Spring Vault 2.0.3 RELEASE

  • Spring Vault 1.1.3 RELEASE
