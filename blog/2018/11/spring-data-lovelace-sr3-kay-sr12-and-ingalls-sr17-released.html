Spring Data Lovelace SR3、Kay SR12和Ingalls SR17 同时发布

release 
rain
2018-11-27
评论

Spring Data 团队宣布发布 Lovelace SR3、Kay SR12和Ingalls SR17 等维护版本。所有这些版本分别获取最新的 Spring Framework 维护版本：5.1.3、5.0.11和4.3.21。Spring Boot 接下来发布的版本（2.1.1、2.0.7、1.5.18）将分别选取 Lovelace SR3、Kay SR12和Ingalls SR17 以便于使用。

该服务发布主要包含错误修复和一些依赖性升级，总共修复了 100 多个问题。您可以通过以下链接找到完整的问题列表：

  • Lovelace SR3

  • Kay SR12

  • Ingalls SR17
