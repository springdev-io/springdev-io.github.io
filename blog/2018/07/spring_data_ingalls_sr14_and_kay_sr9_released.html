Spring Data Ingalls SR14和Kay SR9发布了

release 
rain
2018-07-27
评论
概要
Spring Data Ingalls SR14和Kay SR9发布，属于修复BUG版本。

Kay SR9在刚发布的Spring Framework 5.0.8之上发布，Spring Boot 2.0.4将会更新到此版本，而Ingalls SR14将会被选人Spring Boot 1.5.15中。

两个版本都是修复了一些Bug和升级一些第三方依赖，具体来说，Ingalls SR14关闭了28问题单，而Kay SR9关闭了59问题单。
