Spring Boot 1.5.15现已可用

release 
rain
2018-07-30
评论

Spring Boot 1.5.15包含了35个修复、改进和依赖项更新。

其中修复的部分BUG

  • Flyway文件系统前缀位置检查

  • 当Jetty的WebAppContext无法启动时，应用程序仍然启动成功

  • 由于BeanTypeRegistry持有过时信息，重写的bean定义会导致错误的Bean条件判断

  • 在springProfile的name属性中使用时，不会替换属性占位符

  • 自动配置的MultipartConfigElement会阻止CommonsMultipartResolver解析请求部分

  • 如果Undertow启用了访问日志记录，当容器停止时，线程会泄漏

依赖升级

  • Spring Data Ingalls SR14

  • Spring Amqp 1.7.9.RELEASE

  • Dependency Management Plugin 1.0.6.RELEASE

  • Httpclient 4.5.6

  • Httpcore 4.4.10

  • Tomcat 8.5.32

  • Narayana 5.5.32.Final

  • Maven Enforcer Plugin 1.4.1

  • Git Commit Id Plugin 2.2.4
