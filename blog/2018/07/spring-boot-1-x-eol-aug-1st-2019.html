Spring Boot 1.x 在一年后将停止更新

release 
rain
2018-07-30
评论
概要
今天，Spring Boot的合伙创始人Phil Webb在官方博客正式宣布了Spring Boot 1.x将在2018年8月1号后的一年后停止维护。Spring Boot 1.5分支从1.5.1发布到现在总共发布了15次， 正好经历了18个月。

以下是其全文内容：

所有好东西都必须结束，对于1.x系列的Spring Boot，这意味着我们将需要在今天开始的12个月后，也就是在2019年8月1日停止维护。

我们将继续发布偶尔的1.5.x维护版本，直到那时为止，然后结束分支。

现有的Spring Boot 1.x用户应该相应地进行规划，以确保它们在该日期之前已升级到最新的2.x版本。另请注意，Spring Boot 2.0及更高版本需要Java 8+，因此如果你运行的是Java 6或7，则还应该计划升级JDK。

有关将Spring Boot 1.5应用程序升级到2.0需要执行的操作的详细说明，请查看迁移指南。

祝你好运！
