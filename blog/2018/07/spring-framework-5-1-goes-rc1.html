Spring Framework 5.1 RC1发布了

release 
rain
2018-07-26
评论
概要
Spring Framework 5.1发布第一个RC，带来了诸多新特性。

Spring Framework 5.1 需要JDK 8及以上版本，特别是支持JDK 11这个LTS版本。官方推荐升级到5.1版本，以便使用到JDK 11的新特性，但是鉴于国内项目仍然在大量使用JDK 8，所以这仍然需要一个过程。除此之外，该版本还支持GraalVM，这是Oracle四月份宣布开源的一个支持多语言的高性能虚拟机，目前在开源社区已经大量测试使用。

5.1版本中ObjectProvider被重新设计后，实现了Iterable接口，提供对Stream支持。

Spring Framework 5.1升级到Reactor Californium，支持HTTP2，升级Hibernate ORM到5.3，将使用Hibernate的新SPI，其最新版本5.3.3已经在前天发布。在Web MVC方面主要是优化了日志和异常处理，这对于开发来讲绝对是个好事。

开发团队计划在八月份发布5.1 RC2，主要是一些完善和性能优化工作，GA版本定在9月中旬发布。好消息是Spring Boot 2.1 M1将会在下周发布，这样就可以在Spring Initializr使用到Spring Framework 5.1 RC1！
