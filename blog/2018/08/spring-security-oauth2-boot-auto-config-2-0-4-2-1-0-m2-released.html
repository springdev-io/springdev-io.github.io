Spring Security OAuth2 Boot Auto-config 2.0.4 和 2.1.0.M2 同时发布

release 
rain
2018-08-29
评论

Spring Security OAuth2 Boot Auto-config 2.0.4 和 2.1.0.M2 同时发布，这两个版本主要提供错误修复和依赖版本更新以及一些小的改进。特别值得注意的是，这些与最新版本的Spring Boot保持一致。

该项目来源于Spring Boot 1.x使用的与Spring Security OAuth集成需要的自动配置。由于Spring Security 5对OAuth已经很好的支持，因此这个支持已被删除。为了方便迁移，该项目作为旧的Spring Security OAuth支持和Spring Boot 2.x之间的桥梁而存在。

如果您的项目已经升级到Spring Boot 2.0，请使用Spring Security 5提供的OAuth2支持功能。
