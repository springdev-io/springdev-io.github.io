Spring Cloud Skipper 1.1.0.M1 发布

release 
rain
2018-08-09
评论

Spring Cloud Skipper 1.1.0.M1 发布了。

Spring Cloud Skipper是一个轻量级工具，允许你发现Spring引导应用程序并在多个云平台上管理它们的生命周期。你可以单独使用Skipper或将其与持续集成管道集成，以帮助实现持续部署实践。

参考指南中的入门部分是开始使用的最佳入口。

专注于Cloud Foundry清单:

这个里程碑版本添加了一个新特性，可以执行生命周期操作，在任何可以使用Cloud Foundrymanifest格式部署到Cloud Foundry的应用程序上安装、升级、回滚和删除。

为此，我们将引入一个名为CloudFoundryApplication的新“kind”作为规范，它可以包括这里提到的Cloud Foundry清单条目。可以在安装/升级操作期间重写这些清单项。请查看这些示例，了解更多关于如何在Skipper中管理基于Cloud Foundry清单的包的信息。
