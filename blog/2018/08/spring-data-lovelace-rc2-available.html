Spring Data Lovelace RC2 发布

release 
rain
2018-08-20
评论

Spring Data的项目负责人Oliver Gierke宣布了Spring Data Lovelace RC2 发布，该版本共关闭了82个问题单。你可以与即将发布的Spring Boot 2.1 M2一起使用，正式版需要等待9月中旬。

本次发布了一个非常重要的新功能：Spring Data JPA的存储库延迟初始化，虽然来得有些晚。通过启用Spring Framework的JPA后台初始化选项，可以使你的应用启动时间显著缩短。我们创建了一个示例应用程序，包括了2000个JPA Entity、Repository以及2000个Service并引用到Repository，对比测试后发现，启动时间从原来的35s缩短为23s，当设置启动模式为LAZY时，启动时间可以达到13s。

由于目前已经接近GA版本，如果你想在Spring Boot 2.0应用程序中升级到Lovelace来的话，那么最简单的方法是将spring-data-releasetrain.version属性设置为Lovelace-RC2。

<dependencyManagement>
  <dependencies>
    <dependency>
      <groupId>org.springframework.data</groupId>
      <artifactId>spring-data-releasetrain</artifactId>
      <version>Lovelace-RC2</version>
      <scope>import</scope>
      <type>pom</type>
    </dependency>
  </dependencies>
</dependencyManagement>

除此之外，Lovelace还将首次纳入Spring Data JDBC，该模块集成了MyBatis，实现了CrudRepository，支持@Query、ID生成、事件机制、审计等特性。
