Spring Boot for Apache Geode 和 Pivotal GemFire 1.0.0.M2 发布

release 
rain
2018-08-31
评论

Spring Boot for Apache Geode 和 Pivotal GemFire 1.0.0.M2 发布。

总的来说，这个版本带来了以下改进：

  • 支持Spring Session的自动配置，当使用Apache Geode 或者 Pivotal GemFire / Pivotal Cloud Cache (PCC)来管理Spring Boot应用程序的 (HTTP) 会话状态

  • 使用Spring Boot自动配置的Spring缓存抽象时，可识别spring.cache.type

  • 对文档的补充和改进

  • 升级Spring Framework 5.0.8.RELEASE，Spring Boot 2.0.4.RELEASE 和 Spring Data Kay-SR9

此版本基于最近推出的Spring Session for Apache Geode/Pivotal GemFire (SSDG) 2.0.5.RELEASE构建。

SSDG的一个关键附加功能是使用Properties和一个新的SpringSessionGemFireConfigurer回调接口，来启用Apache Geode / Pivotal GemFire / PCC 动态配置Spring Session。这样，即使在Spring Boot的自动配置环境中，用户也可以自定义配置以满足其应用程序要求。

使用属性配置（Configuration with Properties）

In addition to the Properties discussed here and documented here, Spring Boot for Apache Geode & Pivotal GemFire additionally recognizes the following properties:

除了文档中配置属性之外，Apache Geode和Pivotal GemFire的Spring Boot还识别以下属性：

  • spring.session.timeout

  • server.servlet.session.timeout

同样，如果设置了这些属性的任何组合的话，spring.session.data.gemfire.session.expiration.max-inactive-interval-seconds将优先于spring.session.timeout，它又优先于server.servlet.session.timeout。

Spring Boot for Apache Geode/Pivotal GemFire 能够识别spring.session.store-type。但是，遗憾的是，此属性也无法设置成gemfire 或者 geode，因为这些值不能被Spring Boot识别成为有效的StoreType。

使用Configurer配置（Configuration using a Configurer）

Spring Boot for Apache Geode & Pivotal GemFire 除了Spring Session for Apache Geode/Pivotal GemFire针对新的SpringSessionGemFireConfigurer回调接口提供之外，没有其他任何内容，正如此处所讨论和记录的那样。

下一步

接下来，Spring Boot for Apache Geode & Pivotal GemFire 将提供OOTB HealthIndicators，用以评估和监控Apache Geode / Pivotal GemFire驱动的Spring Boot应用程序的运行时健康状况。

有关 Spring Boot HealthIndicators的更多内容可以查看这里。

这将在下一版本1.0.0.M3中提供。

长远的规划将会支持Spring Boot Actuator。于此同时，也正在努力用Micrometer改进Apache Geode / Pivotal GemFire的统计架构。这将作为Spring Boot for Apache Geode和Pviotal GemFire的Actuator支持的基础。这很可能在SBDG 1.1中提供。

如何使用

您可以通过Spring libs-milestone 库来获取新包，通过在Maven中增加以下依赖项：

  • org.springframework.geode:spring-geode-starter （Apache Geode）

  • org.springframework.geode:spring-gemfire-starter （Pivotal GemFire/PCC）
