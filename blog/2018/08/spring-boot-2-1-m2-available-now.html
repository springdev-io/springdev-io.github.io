Spring Boot 2.1.0 M2 发布

release 
rain
2018-08-21
评论

Spring Boot 2.1的第二个里程碑已经发布，可以从里程碑库中获取。此版本关闭了75个问题和PR。

第二个里程碑建立在Spring Framework 5.1.RC2、Spring Data Lovelace RC2和Spring Security 5.1 RC1等最新的版本之上，并且开始支持Java 11。

其他亮点还包括：

  • 支持任务执行（@EnableAsync）和任务调度（@EnableScheduling）的自动配置

  • 可配置JPA启动模式，极大缩短启动时间

  • 支持OAuth2资源服务器

  • 改进了Web应用程序的日志输出

  • 改进DataSize属性的设置，可以写成10MB
