Spring Framework 5.1 RC2 发布

release 
rain
2018-08-17
评论

Spring Framework 5.1 发布第二个RC，继续开发更多新特性和优化工作。

Spring Framework 5.1 需要JDK 8及以上版本，特别是支持JDK 11这个LTS版本。官方推荐升级到5.1版本，以便能够使用JDK 11的新特性。除此之外，该版本还支持GraalVM，这是Oracle四月份宣布开源的一个支持多语言的高性能虚拟机，目前在开源社区已经大量测试使用。

Spring Framework 5.1 RC2 升级了依赖：Reactor Californium M2、RxJava 2.2和JUnit 5.3 RC1。其中解决了有关支持JDK 11 和GraalVM的几个问题，同时继续完成5.1版本的新特性开发，包括改善日志输出和优化启动性能。

开发团队计划在九月上旬发布5.1 RC3，将会集成Reactor Californium RC1，并改进JDK 11 的支持，GA版本将会在9月底发布。Spring Boot 2.1 M2将会在下周发布，这样就可以在Spring Initializr使用到Spring Framework 5.1 RC2！
