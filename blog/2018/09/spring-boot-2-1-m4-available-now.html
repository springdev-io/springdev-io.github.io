Spring Boot 2.1.0 M4 发布

release 
rain
2018-09-25
评论

Spring Boot 2.1的第四个里程碑已经发布，可以从里程碑库中获取。此版本关闭了40个问题和PR。

第四个里程碑建立在Spring Framework 5.1 GA、Spring Data Lovelace-RELEASE、Spring Security 5.1.0.RELEASE和Reactor Californium等最新的版本之上。

其他亮点还包括：

  • 支持Micrometer的自动配置

  • 支持SAP Hana数据库

  • 支持Spring Data JDBC

  • 自动配置jOOQ ExecutorProvider

  • 优化GC

其中一些主要的第三方依赖升级版本如下：

  • Spring Framwork 5.1.0.RELEASE

  • Spring Data Lovelace-RELEASE

  • Reactor Californium RELEASE

  • Spring Security 5.1.0.RELEASE

  • Spring Kafka 2.2.0.RC1

  • Spring Batch 4.1.0.RC1

  • Spring AMQP 2.1.0.RC1

  • Spring Session Bean-RC1

  • Spring Integration 5.1.0.RC1

  • Micrometer 1.1.0 M1

  • Mongo Driver Reactivestreams 1.9.2

  • Mongodb 3.8.2

  • Jooq 3.11.5

  • Kotlin 1.2.70

  • Influxdb Java 2.13

  • Elasticsearch 6.4.1

  • Htmlunit 2.33

  • Lettuce 5.1.0.RELEASE

  • Hazelcast 3.10.5

开发团队计划在10月中旬发布候选版本，并在10月底发布2.1 GA版本。
