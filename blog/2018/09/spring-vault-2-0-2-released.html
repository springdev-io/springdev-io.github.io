Spring Vault 2.0.2 发布

release 
rain
2018-09-10
评论

Spring Vault提供熟悉的Spring抽象和客户端支持，用于访问、存储和撤销机密。它提供了与Vault交互的低级和高级抽象，使用户免于底层接口问题。

通过HashiCorp的Vault，你可以在所有环境中为应用程序管理外部机密数据。Vault可以管理静态和动态机密，例如应用程序数据，远程应用程序/资源的用户名/密码，并为外部服务提供凭据，如MySQL、PostgreSQL、Apache Cassandra、Consul、AWS等。

Spring Vault 2.0.2 已发布，该版本主要是解决部分缺陷、升级第三方依赖以及优化。

第三方依赖升级如下：

  • Spring Framework 5.0.9

  • Spring Security 5.0.7

  • Spring Data Kay SR10

  • Reactor Bismuth SR11

  • Netty 4.1.29.Final

  • AWS SDK 1.11.404

  • Apache HttpClient 4.5.6

  • HttpCore 4.4.10

  • Jackson 2.9.6

  • Vault 0.10.3
