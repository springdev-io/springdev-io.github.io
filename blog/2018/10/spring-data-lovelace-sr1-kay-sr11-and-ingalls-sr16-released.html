Spring Data Lovelace SR1、Kay SR11和Ingalls SR16 同时发布

release 
rain
2018-10-15
评论
目录
  • Lovelace SR1
  • Kay SR10
  • Ingalls SR16

Spring Data团队宣布推出Lovelace SR1，Kay SR11和Ingalls SR16维护版本。所有这些版本分别获取最新的Spring Framework维护版本：5.1.1、5.0.10和4.3.20。Spring Boot将分别通过其Spring Boot 2.0.6、1.5.17版本获取Kay SR11和Ingalls SR16以便于使用。

该服务发布主要包含错误修复和一些依赖性升级，总共修复了70多个问题。您可以通过以下链接找到完整的问题列表：

  • Lovelace SR1

  • Kay SR11

  • Ingalls SR16
