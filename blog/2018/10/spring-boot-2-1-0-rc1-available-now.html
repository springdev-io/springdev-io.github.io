Spring Boot 2.1.0 RC1 发布

release 
rain
2018-10-17
评论

Spring Boot 2.1 RC1 已经发布，可以从里程碑库中获取。此版本关闭了100多个问题和PR。这次发布作为第一个候选版本，也意味着在最终的2.1 GA版本发布之前，不会添加任何主要的API更改或新功能。正式版预计在月底发布。

我们从之前的里程碑中精选了一部分内容，并提供了一些值得注意的新功能，包括：

  • 其他Micrometer注册表的自动配置：AppOptics，Humio和KairosDB

  • 支持Jersey指标（线程池和HTTP请求）

  • 与Hibernate的BeanContainer API集成

其中一些主要的第三方依赖升级版本如下：

  • Spring Framework 5.1.1.RELEASE

  • Spring Integration 5.1.0.RC2

  • Spring Security 5.1.1.RELEASE

  • Spring Data Lovelace SR1

  • Spring Amqp 2.1.0.RELEASE

  • Reactor Californium-SR1

  • Byte Buddy 1.9.1

  • Commons Lang3 3.8.1

  • Couchbase Client 2.7.0

  • Dropwizard Metrics 4.0.3

  • Ehcache3 3.6.1

  • Elasticsearch 6.4.2

  • Embedded Mongo to 3.5.5

  • Flyway 5.2.0

  • Groovy 2.5.3

  • Glassfish Jaxb 2.3.1

  • Hazelcast 3.10.6

  • Hibernate 5.3.7.Final

  • Infinispan 9.4.0.Final

  • Influxdb Java 2.14

  • Javax Jaxws 2.3.1

  • Kotlin 1.2.71

  • Lettuce 5.1.1.RELEASE

  • Micrometer 1.1.0-rc.1

  • Mockito 2.23.0

  • Netty Tcnative 2.0.17.Final

  • Saaj Impl 1.5.0

  • Sendgrid 4.3.0

  • Sqlite Jdbc 3.25.2

  • Selenium Htmlunit 2.33.0

此版本还一并解决了最近刚刚发布的2.0.6版本已关闭的97个问题。

有关更改和升级说明的完整列表，请参阅WIKI上的Spring Boot 2.1.0.RC1发行说明。

要开始使用Spring Boot 2.1.0 RC1并研究这些新功能，您可以在start.spring.io上轻松创建新项目。
