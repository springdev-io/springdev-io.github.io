Spring Cloud Function 2.0.0.RC1 发布

release 
rain
2018-10-29
评论

Oleg Zhurakousky在博客宣布了Spring Cloud Function 2.0.0.RC1 发布。

Spring Cloud Function 2.0.0.RC1 作为独立的模块可以在Spring Milestone仓库获取到。

除了一般增强功能和错误修复，此版本的主要主题是：

  • 支持Kotlin

  • 为Azure函数提供的新编程模型

  • 在Web适配器启动时自动（可选）导出Supplier

  • 函数式Bean注册

  • 支持输入类型为Collection的函数的透明类型转换

  • 构建在Spring Boot 2.1上
