Spring Tools 4.0.1 发布

release 
rain
2018-10-31
评论

Spring Tools 4是您喜爱的编码环境的下一代Spring工具。已经完全从头开始重写，为开发基于Spring的企业应用程序提供世界级的支持，无论您喜欢Eclipse、Visual Studio Code还是Atom IDE。

Spring Tools 4 for Eclipse、Visual Studio Code和Atom的4.0.1版本已经发布了。

此版本的亮点包括有：

  • Eclipse 新增支持 Buildship 3.0

  • Eclipse 分发再次包括用于Apache Tomcat的服务器适配器

  • Concourse 更新了S3参数到最新变更

  • Spring Boot 相关的一些缺陷修复

要下载Eclipse的发行版，以及找到Visual Studio Code和Atom的应用市场的链接，请访问：

  • Spring Tools 4: https://spring.io/tools/

详细的更新说明查看这里：https://github.com/spring-projects/sts4/wiki/Changelog#2018-10-31-401-release
