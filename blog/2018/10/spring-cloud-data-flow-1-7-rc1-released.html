Spring Cloud Data Flow 1.7 RC1 发布

release 
rain
2018-10-19
评论

Spring Cloud Data Flow团队宣布Spring Cloud Data Flow 1.7 RC1 发布。请按照Local Server，Cloud Foundry和Kubernetes的入门指南进行操作。

RC1版本基于1.7 M1中引入的核心功能，并进行了一些改进。

以下是主要内容：

  • 改进的UI——UI具有全新的外观。如果白名单为空，则属性白名单功能已经过细化，默认情况下不会显示所有应用程序属性。

  • 流应用程序DSL——这已集成到UI中，因此您可以设计由任意Boot应用程序构成的数据管道。

  • 支持在Kubernetes上调度任务

  • 审计跟踪（Audit trail）——任务和计划程序已支持

  • 并发任务启动限制（Concurrent Task Launch Limiting）

  • 流和任务校验——添加了对“app”类型注册的应用程序验证的支持

  • 强制升级流应用
