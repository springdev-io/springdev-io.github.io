Spring Cloud Open Service Broker 3.0.0.M1 已发布

release 
rain
2018-10-03
评论

Spring Cloud Open Service Broker是一个用于构建实现Open Service Broker API的Spring Boot应用程序的框架。

Open Service Broker API项目允许开发人员为云原生平台（如Cloud Foundry，Kubernetes和OpenShift）中运行的应用程序提供服务。Spring Cloud Open Service Broker提供了一个基于Spring Boot的框架，使你能够在支持Open Service Broker API的平台上为你自己的托管服务快速创建服务代理。

Spring Cloud Open Service Broker，目前需要Java 8、Spring Framework 5、Spring Boot 2，支持Spring MVC或者Spring WebFlux风格的应用，提供了Spring Boot的自动配置。

Spring Cloud Open Service Broker 3.0.0.M1 已发布。3.0版本以2.1.0.M1为基础，并引入了完整的响应式支持，该版本将支持Open Service Broker API v2.14规范中的新增和变更内容。

此版本包括以下内容：

  • 为Spring WebFlux和Project Reactor提供一流支持

  • starter已重命名，因为它现在支持两个Spring Web框架

  • 引入几个响应式生命周期钩子，用于执行附加操作

包括以下Spring Boot 起步依赖：

<dependency>
  <groupId>org.springframework.cloud</groupId>
  <artifactId>spring-cloud-starter-open-service-broker</artifactId>
  <version>3.0.0.M1</version>
</dependency>

使用Spring Milestone库：

<repository>
  <id>spring-milestones</id>
  <url>https://repo.spring.io/milestone</url>
</repository>

在GitHub查看已关闭问题单。在GitHub上查看已关闭的问题列表。下一个版本3.0.0.M2将包括对异步绑定的支持。请密切关注接下来几周的进展情况。
