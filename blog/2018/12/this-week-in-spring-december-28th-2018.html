Spring一周回顾（2018年12月28日）

news 
rain
2018-12-28
评论

大家好，Spring 粉丝们，感谢一如既往的等待，又到了每周必读的《Spring一周回顾》时间。

2019 年马上就快要到了，祝大家新年快乐，心想事成，财源滚滚，好运连连！

经过两周的精心准备，我们已经编译整理好《Spring 2018 年终报告》和《Spring 2018 文章精选合集》，这是给技术社区的一个阶段总结，也是感谢这一年来大家的关注和支持，无以为报，仅此而已。 大家可以关注 “Spring技术社区”，然后发送关键词 “年终报告” 和 “文章精选合集” 分别获取相应的电子版（PDF、EPub、Mobi 等格式）。

由于大家的积极参与，Spring 年终调查也有了结果，再次感谢大家！

图 1. Spring Boot 使用版本调查结果
图 2. Spring Cloud 使用版本调查

本周我为大家带来了如下内容：

  • Spring Session for Apache Geode/Pivotal GemFire 2.1.2.RELEASE 发布

  • Apache Groovy 2.5.5 已发布

  • Apache Flink 1.5.6 已发布

  • Apache Log4j Kotlin API 1.0.0 已发布

  • Apache NetBeans 10.0 已发布

  • Node v11.6.0、v10.15.0、v8.15.0、v6.16.0 已发布

  • Ruby 2.6.0 已发布

  • Python 3.7.2 已发布、3.6.8 已发布

在知乎上面，我回答了两个初学者的学习 Spring 相关的问题，也许对大家有所帮助：

  • Spring入门该学习什么？

  • 初学Spring有没有适合的书？

历史好文推荐：

  • 上一期的《Spring 一周回顾》同样精彩，值得一读！

  • 《Spring 开发指南，带你进入 Spring 的世界》

  • 《Spring 最全书单，送给爱学习的你》，80 本国内外已出版的 Spring 相关的书，可以根据自己需要选择阅读！

  • 《10 种保护 Spring Boot 应用程序的绝佳方法》，来自业界安全专家和开源大咖 Matt Raible 和 Simon Maple 合作的成果。

  • 《16 条 Spring Boot 最佳实践》，来自业界 Spring 大咖们的精彩总结！

节日愉快，感谢陪伴，我们明年见！
