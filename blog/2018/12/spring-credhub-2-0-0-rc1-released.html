Spring CredHub 2.0.0.RC1 发布

release 
rain
2018-12-14
评论

Scott Frederick 在博客上宣布了 Spring CredHub 2.0.0 第一个 RC 版本发布, 可以从 Spring 里程碑版本库获取。Spring CredHub 2.0.0 GA 版本即将在几周后发布。

介绍 CredHub

CredHub 为 Cloud Foundry 平台提供凭证的集中管理。CredHub 实现了一个 HTTP API 来安全地存储、生成、检索和删除各种类型的凭证。

Spring CredHub 为 CredHub API 提供了一个 Java 绑定实现，可以很容易地将 Spring 应用程序与 CredHub 进行集成。

有什么新的？

  • 2.0.0 版本支持 CredHub 服务器版本 1.x 和 2.x，以及 CredHub v1 和 v2 API

  • 提供与 Spring Framework 5 和 Spring Boot 2 的兼容性

  • 借助 Reactor 库添加了对现有接口的响应式实现。

入门

Spring CredHub 文档包含一个入门部分，可帮助您开始使用该库，可以通过 GitHub 问题提供反馈。
