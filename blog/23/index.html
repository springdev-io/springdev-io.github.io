Spring Session Apple-SR4 和 Bean-M1 同时发布

Spring Session使得支持会话集群变得轻而易举，同时不依赖于特定应用程序容器的解决方案。它提供了针对HttpSession、WebSocket、WebSession的无缝集成。目前已经集成并实现了JDBC、Hazelcast、Redis、MogoDB、Geode和GemFire等多种存储方案。

Spring Session Apple-SR4 和 Bean-M1 两个版本同时发布。Spring Boot用户可以在已经发布的2.0.4.RELEASE 和 2.1.0.M1 版本中选择使用。
