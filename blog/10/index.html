Spring Session Bean GA 已发布，有什么新特性，版本如何选择？

release 
rain
2018-10-31
评论
Spring Session 新发布的Bean有什么新特性？项目中如何选择合适的版本？

Spring Session Bean 版本的新特性

Spring Session BOM Bean GA 现已发布，这是基于Spring Session 2.1的第一个版本，可以很容易地与新发布的Spring Boot 2.1一起使用。

Same-Site Cookie支持与HttpSession集成

Same-Site Cookie是另一种帮助开发人员防止跨站点请求伪造的机制。我们的DefaultCookieSerializer已得到增强，支持将SameSite属性添加到Spring Session生成的会话cookie中。 默认情况下，SameSite属性启用，值为Lax，可使用DefaultCookieSerializer#setSameSite自定义。

请注意，从Spring Framework 5.1开始，Spring WebFlux本身就已有对WebSession的同样支持。

HttpSessionBindingListener的支持

Spring Session现在支持HttpSessionBindingListener，并且会在此API的实现上正确调用回调。这也意味着现在可以在会话作用域上使用@PreDestroy，因为该功能是使用Spring Framework的DestructionCallbackBindingListener实现的，这是HttpSessionBindingListener的具体实现。

支持与Redis WebSession集成可自定义序列化实现

我们的Redis WebSession集成现在可以轻松配置为使用自定义序列化机制。以类似于Redis HttpSession集成的方式，可以通过提供名为springSessionDefaultRedisSerializer的类型为RedisSerializer<Object>的@Bean来实现：

@EnableRedisWebSession
static class SessionConfig {

    @Bean
    public RedisSerializer<Object> springSessionDefaultRedisSerializer() {
        // configure and return a serializer of your choice
    }

}

JDBC HttpSession集成的性能得到改进

我们为JDBC HttpSession集成做了一些性能改进：

  • 会话属性的延迟反序列化 - 会话属性现在将在第一次访问时反序列化，而不是从数据库中检索HttpSession，这可以为许多HTTP请求完全避免反序列化的成本

  • 禁用事务的选项 - 对于想要避免事务成本的用户，JdbcOperationsSessionRepository现在提供一个只接受JdbcOperations并在内部使用no-op TransactionOperations实现的构造函数

支持 Java 11

Spring Session现在支持最近发布的Java 11，而所需的版本当然还保留在Java 8。我们的CI构建已得到增强，因此该项目现在可以针对Java 11进行持续验证。

第三方依赖升级

Spring Session 2.1基于以下最新和最高版本的关键依赖项：

  • Spring Framework 5.1

  • Spring Data Lovelace

  • Spring Security 5.1

  • Project Reactor Californium

release 
rain
2018-10-30
评论
Spring Cloud for Alibaba 0.2.0 发布，这是作为孵化项目的首次发布版本。

大家好，很高兴地告诉大家，今天 Spring Cloud Alibaba 的第一个版本发布了。

Spring Cloud Alibaba 项目由两部分组成：包含了阿里巴巴的开源组件和阿里云的产品。旨在利用众所周知的 Spring 框架的设计模式和抽象能力，以便为 Java 开发人员使用阿里巴巴产品带来 Spring Boot 和 Spring Cloud 的优势。

Spring Cloud Alibaba 项目是由阿里巴巴维护的社区项目。

注意： 版本 0.2.0.RELEASE 对应的是 Spring Boot 2.x 版本，版本 0.1.0.RELEASE 对应的是 Spring Boot 1.x 版本.
