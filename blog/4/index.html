Spring Security 5.1.2、5.0.10和4.2.10 同时发布，这些版本主要提供错误修复和依赖版本更新以及一些小的改进。

Spring Security是一个功能强大且可高度自定义的身份验证和访问控制框架。它是保护基于Spring的应用程序的事实上的标准。

其特性如下：

  • 对身份验证和授权的全面和可扩展的支持

  • 防止会话固定，点击劫持，跨站点请求伪造等攻击

  • Servlet API集成

  • 支持与Spring Web MVC集成
