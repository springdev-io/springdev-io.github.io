Spring Security 5.1.4 发布

Spring Security 是一个功能强大且可高度自定义的身份验证和访问控制框架。它是保护基于 Spring 的应用程序的事实上的标准。

其特性如下：

  • 对身份验证和授权的全面和可扩展的支持

  • 防止会话固定，点击劫持，跨站点请求伪造等攻击

  • Servlet API集成

  • 支持与Spring Web MVC集成

Spring Framework 5.1.5 发布

release 
rain
2019-02-13
评论
Spring Framework 5.1.5 发布，解决 40 多个缺陷和改进。

Spring Framework 5.1.5 已发布，解决 40 多个缺陷和改进。

  • 修复了在与 '*/' 在同一行的多行注释中出现 '-' 时 ScriptUtils 失败的问题

  • InjectionPoint 自动装配会由于通过名称方式来自动装配的 @resource bean 而引发异常

  • PathMatchingResourcePatternResolver 可以双重包装 jar：URL

  • mariadb-java-client 2.4.0 productName 已更改导致不兼容 Spring Batch

  • SpringEL 不应该为无效赋值抛出 IllegalAccessError

  • 避免在 DefaultListableBeanFactory.findAnnotationOnBean 中重复调用 findAnnotations

  • 支持 WildFly 13+ 的加载时织入

  • org.springframework.web.client.HttpMessageConverterExtractor＃extractData 在缺少内容长度标头时无法检测到空体

  • 弃用 JibxMarshaller

  • 无法禁用 DefaultExceptionHandler 日志警告

  • 在 Jackson2JsonDecoder 中支持 null 文字

Micronaut 1.1.0 M1 发布，模块化、支持 gRPC。

来自 Object Computing 的 Micronaut 开发团队宣布了 Micronaut 1.1.0 M1 已经发布。

Micronaut 自开创性的 1.0 版本以来，1.1 包含了许多重要的改进，包括有以下几个主要内容：

  • 支持 GRPC

  • 支持 AWS API Gateway 和 GraalVM 自定义运行时

  • 支持 AWS Alexa Lambda

  • 本机文件监视和快速服务器重启

  • Micronaut Test 的新测试模板

  • 更快的冷启动时间和性能优化

  • 更多新特性和改进…​

Micronaut 1.1 的下一个也是最后一个里程碑将在发布候选版开始之前包括支持 RabbitMQ 和 OpenID Connect。
