Spring Cloud Task 2.1.0.M1 发布

release 
rain
2018-11-05
评论
Spring Cloud Task 2.1.0.M1 发布，支持Spring Boot 2.1.0。

Spring Cloud Task 2.1.0M1 现已发布，可以在 Github 和 Pivotal 下载库中找到。

更新了什么？

什么是新的？

Spring Cloud Task 2.1.0.M1 旨在成为与Spring Boot 2.1.0一致的框架版本。

相比较2.0.x，此版本的更新包括：

  • 更新所有依赖项

  • Spring Cloud Task 现在通过自动配置启用

  • 执行任务时，TaskExecution的exitCode为null

  • Spring Cloud Task 在Java 8、9、10、11上编译和运行

让我们更详细地介绍这些更新。

更新所有依赖项

如前所述，此版本使Spring Cloud Task与Spring Boot 2.1保持一致。这包括更新所有依赖项。

Spring Cloud Task 现在通过自动配置启用

再见@EnableTask，你好自动配置！Spring Cloud Task现在通过自动配置启用。要使用它，请将spring-cloud-starter-task依赖项添加到Maven pom或Gradle构建文件中。

执行任务时，TaskExecution的exitCode为null

在Spring Cloud Task的早期版本中，从Spring Cloud Task方法返回的TaskExecution总是显示正在运行的任务的exitCode为0。现在，如果任务仍在执行，则从Spring Cloud Task返回的TaskExecutions的exitCode为null。
