Spring Data Lovelace SR6、Kay SR14和Ingalls SR20 同时发布

Spring Data 团队宣布发布 Lovelace SR6、Kay SR14和Ingalls SR20 等维护版本。所有这些版本分别获取最新的 Spring Framework 维护版本：5.1.6、5.0.13和4.3.23。Spring Boot 接下来发布的版本（2.1.4、2.0.9、1.5.20）将分别选取 Lovelace SR6、Kay SR14和Ingalls SR20 以便于使用。

此次版本发布包含有建议的安全更新，解决了 CVE in Spring Data JPA。

  • Spring Integration 4.3.20

  • Spring Retry 1.2.4

  • Spring Security 4.2.12
