Spring CredHub 2.0.0.RELEASE 发布

Scott Frederick 在博客上宣布了 Spring CredHub 2.0.0 版本正式发布, 可以从 Spring Release 和 Maven Central 仓库获取到。

介绍 CredHub

CredHub 为 Cloud Foundry 平台提供凭证的集中管理。CredHub 实现了一个 HTTP API 来安全地存储、生成、检索和删除各种类型的凭证。

Spring CredHub 为 CredHub API 提供了一个 Java 绑定实现，可以很容易地将 Spring 应用程序与 CredHub 进行集成。

Spring Data Lovelace SR4、Kay SR13和Ingalls SR18 同时发布

Spring Data 团队宣布发布 Lovelace SR4、Kay SR13和Ingalls SR18 等维护版本。所有这些版本分别获取最新的 Spring Framework 维护版本：5.1.4、5.0.12和4.3.22。Spring Boot 接下来发布的版本（2.1.2、2.0.8、1.5.19）将分别选取 Lovelace SR4、Kay SR13和Ingalls SR18 以便于使用。

该服务发布主要包含错误修复和一些依赖性升级，总共修复了 170 多个问题。您可以通过以下链接找到完整的问题列表：
