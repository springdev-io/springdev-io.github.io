Spring Session for Apache Geode/Pivotal GemFire 2.0.6.RELEASE和2.1.0.RELEASE 同时发布

Spring Data 团队的 John Blum在开发者博客上宣布了 Spring Session for Apache Geode/Pivotal GemFire（SSDG）2.0.6.RELEASE 和 2.1.0.RELEASE 同时发布！

SSDG 2.0.6.RELEASE 基于 Spring Framework 5.0.10.RELEASE、Spring Data Kay-SR11 和 Spring Session core 2.0.7.RELEASE 等核心依赖升级。

SSDG 2.1.0.RELEASE 基于 Spring Framework 5.1.1.RELEASE、Spring Data Lovelace-SR1 和 Spring Session 2.1.0.RELEASE，另外还包括对自定义会话过期策略和规则的支持。
