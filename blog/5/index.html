Spring Boot 2.2 M1 发布

Spring Boot 2.2 M1 已经发布，现在可以从 里程碑版本库中获取。该版本工解决了 超过 140 个问题和合并请求。

第一个里程碑版本的主要亮点有：

  • Spring Data Moore M2

  • 大量配置属性的绑定速度明显加快

  • 对 bean 初始化时懒加载的可选支持（在随后的博客文章中会有更多内容）

  • JMX 默认不启用

  • 大量第三方依赖库升级

  • 使用 Actuator 时，启动速度更快，内存占用更少

Spring Data Moore M2 发布

release 
rain
2019-03-07
评论
Spring Data Moore M2 发布

Spring Data 开发团队的 Christoph Strobl 目前在博客上宣布了 Moore 的第二个里程碑版本已发布。

值得注意的变化包括有：

  • Spring Data MongoDB、Cassandra 和 Redis 中支持 Kotlin 协程（Coroutines）

  • Querydsl 支持响应式存储库

  • 支持在 Spring Data MongoDB & Cassandra 派生的存储库查询方法 between 使用 Range 类型

  • Neo4j 存储库方法支持 exists 投影

  • Spring Data Elasticsearch 提供了 响应式存储库

  • Pivotal Gemfire 和 Apache Geode 版本升级

  • 升级到 Spring HATEOAS 1.0 M1

  • JDBC module 的其他一些新特性，诸如跳过 是否新数据 检查，直接 insert 和 update
