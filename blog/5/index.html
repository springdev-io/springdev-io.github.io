news

Spring 开发者们每周必读的《Spring一周回顾》。

大家好，Spring 粉丝们，感谢一如既往的等待，又到了每周必读的《Spring一周回顾》时间。

大家还记得 15 年前么，也就是 2004 年的 3 月 24 日，Spring Framework 正式对外发布 1.0 版本。 15 年的时间，Spring 从最初两三个人的小团队，到成立一家开源技术和咨询的公司，再到被巨头公司收购。 15 年的时间，Spring 从最初一个框架的单一开源项目，到发展成为一个产品家族系列，从最初的支持 Java 1.4 到现在的 Java 12，同时也是开源技术的集大成者，见证了众多开源技术的兴起到衰落，Spring 已经成为企业级开发中必不可少的技术选择之一，也让众多的 Java 开发者们因此拥有了喜爱的工作，也因此而见证了一代程序员们的成长之路。
