Spring Cloud Open Service Broker 2.1.0 发布

Spring Cloud Open Service Broker 是一个用于构建实现 Open Service Broker API 的Spring Boot 应用程序的框架。

Open Service Broker API 项目允许开发人员为云原生平台（如 Cloud Foundry，Kubernetes 和 OpenShift）中运行的应用程序提供服务。Spring Cloud Open Service Broker 提供了一个基于 Spring Boot 的框架，使你能够在支持 Open Service Broker API 的平台上为你自己的托管服务快速创建服务代理。
