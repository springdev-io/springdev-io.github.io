JHipster v6.1.0 发布

release 
rain
2019-06-11
评论

JHipster 是一个开发平台，用于生成、开发和部署 Spring Boot + Angular/React Web 应用程序和 Spring 微服务。

该框架的目标是为开发者生成一个完整并统一的现代 Web 应用程序或微服务架构：

  • 使用 Spring Boot 在服务器端实现高性能和强大的 Java 技术栈

  • 这是一款时尚、现代、移动优先的前端框架，支持 Angular、React 和 Bootstrap

  • 一个强大的微服务架构，包括 JHipster Registry、Netflix OSS、ELK 和 Docker

  • 使用 Yeoman、Webpack 和 Maven/Gradle 构建应用程序的强大工作流程

这是 JHipster v6 之后的第一次小版本发布，总共关闭了 182 个问题单和代码合并请求。

下面是一些重要的更新：

  • 迁移到 Angular 8 版本 - #9817

  • 改进 Kafka 支持 - #9567

  • 修复复杂区域设置的本地化电子邮件 - #9722

  • 新增语言支持：Finnish - #9843

  • 为关系型数据库提供 UUID 字段类型 - #9794

其中一些第三方依赖库的升级如下：

  • Spring Boot 2.1.5

  • Spring Framework 5.1.7.RELEASE

  • Spring Security Oauth 2.3.6.RELEASE

  • Jhipster Framework v3.0.2

  • Jhipster Registry v5.0.2

  • NodeJS 10.16.0

  • Dropwizard Metrics 4.1.0

  • Hibernate 5.3.10.Final

  • Infinispan 9.4.14.Final

  • Test Containers 1.11.3

  • JIB 1.3.0

  • ng-jhipster 0.10.1

同时，使用到的 Docker 镜像版本也已升级到最新版本：

  • Mongo docker image 4.0.10

  • Grafana docker image 6.2.2

  • Memcached docker image 1.5.16-alpine

  • Mariadb docker image 10.4.5

  • Postgres docker image 11.3

  • Consul docker image 1.5.1

  • Prometheus docker image v2.10.0

  • Prometheus operator docker image v0.30.0

  • Alertmanager docker image v0.17.0

  • Traefik docker image 1.7.12
