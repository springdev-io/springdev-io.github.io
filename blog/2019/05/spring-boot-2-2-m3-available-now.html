Spring Boot 2.2 M3 发布

release 
rain
2019-05-15
评论

Spring Boot 2.2 M3 已经发布，现在可以从 里程碑版本库中获取。该版本工解决了 100 个问题和 PR。

有关更改和升级说明的完整列表，请参阅 Wiki 上的 Spring Boot 2.2 发行说明和 已更新的参考文档。

如果您想开始使用 2.2 并尝试新功能，可以在 https://start.spring.io 上创建一个新项目。

其中一些第三方依赖库升级如下：
