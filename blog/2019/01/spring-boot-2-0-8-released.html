Spring Boot 2.0.8 发布

release 
rain
2019-01-11
评论

Spring Boot 2.0.8 已经发布，共解决了 35 项问题修复、改进和依赖项更新，建议使用 2.0.x 的用户升级到此版本。您可以从 Spring Release 和 Maven Central 下载获取。

其中依赖升级到最新版本：

  • Spring Framework 5.0.12.RELEASE

  • Spring Session Apple-SR8

  • Spring AMQP 2.0.11

  • Spring Batch 4.0.2.RELEASE

  • Spring Security 5.0.11

  • Spring Integration 5.0.11

  • Spring Kafka 2.1.12.RELEASE

  • Spring Restdocs 2.0.3.RELEASE

  • Spring Data Kay SR13

  • Spring Web Services 3.0.6.RELEASE

  • Reactor Bom Bismuth-SR15

  • Elasticsearch 5.6.14

  • Hibernate Validator 6.0.14.Final

  • Micrometer 1.0.9

  • Maven Assembly Plugin 3.1.1

  • Neo4j Ogm 3.1.6

  • Jackson 2.9.8

  • Johnzon Jsonb 1.1.11

  • Jedis 2.9.1

  • Jetty El 8.5.35.1

如果您有兴趣参与开源贡献，请查看问题库中的 “ideal for contribution”标签。如果您有一般性问题，请在 stackoverflow.com 上使用 spring-boot 标签或在 Gitter上与社区其他人聊天。
