Spring一周回顾（2019年1月11日）

news 
rain
2019-01-11
评论

大家好，Spring 粉丝们，感谢一如既往的等待，又到了每周必读的《Spring一周回顾》时间。

本周我为大家带来了如下内容：

  • Spring Boot 2.1.2 发布

  • Spring Boot 2.0.8 发布

  • Spring Boot 1.5.19 发布

  • Spring Security 5.1.3、5.0.11和4.2.11 同时发布

  • Spring CredHub 2.0.0.RELEASE 发布

  • Spring Data Lovelace SR4、Kay SR13和Ingalls SR18 同时发布

  • Spring Framework 5.1.4、5.0.12和4.3.22 同时发布

  • Spring Cloud Function 2.0.0.RELEASE 发布

  • Spring Cloud Stream - Fishtown.RELEASE 发布

  • Apache Subversion 1.9.10、1.10.4、1.11.1 已发布

  • Apache Jackrabbit 2.10.9 已发布

  • Apache HBase 2.1.2 已发布

  • Apache Guacamole 1.0.0 已发布

  • Apache Flume 1.9.0 已发布

  • Hibernate Validator 6.0.14.Final 已发布

  • Atom v1.34.0 已发布

  • Visual Studio Code 1.30.2 已发布

  • Angular 7.2.0 已发布

  • Webpack v4.28.4 已发布

  • Gradle 5.1.1 已发布

  • Serverless 1.36.0 已发布

  • Kubernetes 1.13.2 已发布

  • Play Framework 2.6.21 已发布

  • Micronaut 1.0.3 已发布

历史好文推荐：

  • 上一期的《Spring 一周回顾》同样精彩，值得一读！

  • Spring 技术社区的新年展望，《Spring in 2018 年终报告》和《Spring in 2018 文章精选合集》电子版免费下载！

周末愉快，感谢阅读，我们下周见！
