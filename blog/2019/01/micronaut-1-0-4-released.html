Micronaut 1.0.4 发布

release 
rain
2019-01-29
评论

Micronaut 1.0.4 带来了以下新内容：

修复缺陷和功能改进

  • 修复若干缺陷

  • 功能改进

模块化

Micronaut 现在是模块化的，部分 Micronaut 被转移到子项目中。 请参阅以下链接，了解现在在 Micronaut Core 之外具有独立发布周期的新子项目：

  • Micronaut Kafka

  • Micronaut Micrometer

  • Micronaut Groovy

  • Micronaut SQL

  • Micronaut MongoDB

  • Micronaut Redis

  • Micronaut Test

  • Micronaut Netflix

GraalVM 1.0 RC11 Support

GraalVM 支持已更新，以适应 GraalVM 1.0.0 RC11 中的最新更改。

第三方依赖升级：

  • Kafka 2.0.0 → 2.0.1

  • Reactive Streams 1.0.1 → 1.0.2

  • Jackson 2.9.7 → 2.9.8
