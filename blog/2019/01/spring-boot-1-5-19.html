Spring Boot 1.5.19 发布

release 
rain
2019-01-11
评论

Spring Boot 1.5.19 已经发布，共解决了 17 项第三方依赖库更新。

需要说明的是，由于Spring Boot 1.x 已经宣布进入EOL了，如果您还没有升级，请尽快考虑升级和迁移。

第三方依赖库升级：

  • Spring Framework 4.3.22.RELEASE

  • Spring Data Ingalls SR18

  • Spring Session 1.3.5

  • Spring Security 4.2.11

  • Spring Integration 4.3.19

  • Spring Batch 3.0.10.RELEASE

  • Spring Web Services 2.4.4.RELEASE

  • Spring Retry 1.2.3.RELEASE

  • Spring Amqp 1.7.12.RELEASE

  • Git Commit Id Plugin 2.2.6

  • Narayana 5.5.34.Final

  • Groovy 2.4.16

  • Tomcat 8.5.37

  • Appengine Sdk 1.9.71

  • GemFire 8.2.13

  • Jedis 2.9.1

  • Jetty 9.4.14.v20181114
