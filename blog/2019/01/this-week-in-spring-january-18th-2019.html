Spring一周回顾（2019年1月18日）

news 
rain
2019-01-18
评论

大家好，Spring 粉丝们，感谢一如既往的等待，又到了每周必读的《Spring一周回顾》时间。

“Hello World” 是学习新编程语言时通常编写的第一个程序。在布莱恩·柯尼汉（Brian Kernighan）的 B 编程语言教程中首次提到它后，它通过 Kernighan ＆ Ritchie 1978年出版的 “The C Programming Language” 一书而广为人知，它的内容如下：

main() {
    printf("hello, world\n");
}
图 1. "Hello, world" program by Brian Kernighan (1978)

从那时起，Hello World 已经在地球上的几乎所有编程语言中实现。 在 The Hello World Collection 上有 585 个 Hello World 程序，包括许多知名或者不知名的编程语言，以及 78 种人类语言。除了“发明”了 “hello, world” 之外，他被网友和媒体戴上了各种头衔—— “历史上最伟大的十大程序员”、“十五位计算先驱” 等等，他与丹尼斯·里奇（Dennis Ritchie）共同写作了C语言的第一本著作《C程序设计语言》，当今的程序员们逃不开他的影响力。与高高在上的“大神”形象形成反差的是，在普林斯顿大学任教的布莱恩·柯尼汉教授，每年秋天的开学季，都会给非计算机专业的学生，讲授一门名叫“我们世界中的计算机”的课程（COS 109: Computers in our World）。这门课的目的其实非常纯粹，就是科普有关计算机的最基本常识，让每一个人都能在工作和生活中更好地使用电脑。

在本月的主题内容里，我编写了这个 Spring Boot、Grails、Ratpack 等多个框架的多语言入门版本《Hello World》应用系列，目前已经完成了 Spring Boot 的 Java、Groovy、Kotlin、Scala、Clojure 等版本，后续还会陆续发布其他框架的版本，敬请期待。同时，这个也是初学者非常好的入门学习教程，内容涉及开发工具、创建项目、编写代码、代码讲解、单元测试、集成测试等，看似简单但同样认真对待，力争做到全面、细致，也同时欢迎 Spring 开发者投稿参与，从简单的一行代码开始，在 2019 年有一个美好的、崭新的起点。

之所以要做这个事情，一方面是 Spring Framework 从 2.0 版本开始就支持 Groovy、Ruby 等动态语言，此外还开始了 Spring.NET、Spring Scala、Spring Python 等支持多语言的项目。虽然这些项目后面由于多种原因停止了，但是 JVM 上多语言也一直在持续不断发展，从目前发展的多个语言来看，Kotlin 在 2018 年发展迅速，Spring Framework 从 5.0 对 Kotlin 提供了一流的支持，而 Scala 和 Clojure 语言在大数据处理方面也发展不错。

另一方面，Spring Boot 和 Spring Cloud 在微服务架构中越来越流行，而多语言编程（Polyglot Programming）也是微服务架构中的一个优势，除了利用多语言和多语言平台的优势外，还可以支持技术团队自由选择“合适的工具做正确的事”，这也是非常吸引开发者的地方。
