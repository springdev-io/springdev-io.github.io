Spring 技术社区，2019 新年快乐！

news 
rain
2019-01-01
评论

大家好，Spring 粉丝们，感谢大家的支持，不知不觉，Spring 技术社区已经走过了半年的时间。

Spring 技术社区，在这半年的时间里总共编写了 179 篇文章，其中有关 Spring 项目 有 101 篇，Micronaut 有 7 篇。此外，经过两周的精心整理，我们写了《Spring 2018 年终报告》和《Spring 2018 文章精选合集》两本电子书。

这是给技术社区的一个阶段总结，也是感谢这一年来大家的关注和支持，无以为报，仅此而已。 大家可以关注 “Spring技术社区”，然后发送关键词 “年终报告” 和 “文章精选合集” 分别获取相应的电子版（PDF、EPub、Mobi 等格式）。

其中,

《Spring 2018 年终报告》回顾了 2018 年发生的大家关注的开源社区事件，也汇总和分析了 Spring 开源项目的一些当下项目现状，开源项目除了已经在使用更需要大家持续的关注和更深入的参与、分享和贡献，最后也对下一年做了一些技术思考和新年展望，或许对大家有些许的参考价值。

《Spring 2018 文章精选合集》则是对 2018 年技术社区一些 Spring 一周回顾、版本发布说明、开发指南和文章的精选合集，总共有 240 页。

其中，也有几篇文章非常棒，再次推荐给大家：

  • 《Spring 开发指南，带你进入 Spring 的世界》

  • 《Spring 最全书单，送给爱学习的你》

  • 《10 种保护 Spring Boot 应用程序的绝佳方法》

  • 《16 条 Spring Boot 最佳实践》

  • 《Micronaut 正式开源，构建微服务新选择！》

另外，在 12 月初发布的 Spring 年终调查，经过大家的积极参与，现在也有了结果，再次感谢大家！

2019 年也是非常重要的一年，又会发生哪些事情，哪些是我们期待的。

语言上，Java 11 开始在生产使用，12、13 将发布；Scala 2.13 将会发布，Groovy 3.0 将会 发布，Kotlin 1.4 or 2.0 ? Go 将会发布 2.0。Spring 是否重启对 .NET 的支持？发布 Spring.NET Framework 3.0？

框架上，微服务框架，多语言，多平台，云和容器支持；Serverless、FaaS。 除了 Spring Boot & Cloud 之外，其他厂商对框架的投入，比如 Dubbo、Micronaut，都是非常期待 Spring Cloud 继续加强和云服务提供商（Google、AWS、Azure、阿里云、华为云、腾讯云等）的深度 合作，更多服务集成到框架中。

工具上，IDE 最强之争，战火重新点燃？JetBrains（IntelliJ IDEA、WebStorm 等） vs Eclipse vs VS Code vs NetBeans，目前 VS Code 在 2018 年增长很快，目前着重还是在前端框架和工具的支持上。2019 年会加大 GitHub 整合、云端开发和部署工具等方面的投入。

移动平台上，目前移动市场显然已经不是 Spring 的重点和优势所在，Spring 虽然有 spring-mobile 和 spring-android，但是支持很有限，而且很久没有更新了。

大数据和人工智能方面，目前 Google、Facebook、Microsoft、IBM、阿里巴巴、百度、腾讯等也 都有开源一些项目，都在大力投入和积极布局自己的生态，人才、产品、服务、开源、工具等方面竞争无处不在。 Google 有 TensorFlow，Facebook 有 PyTorch 和 Caffe2，Microsoft 有 CNTK，IBM 有 Watson 和 已开源的 SystemML，云服务提供商可以提供大数据和人工智能的一些中间件产品，使得企业使用很方面能够快速 进入这个领域。Apache 有一些开源框架，比如 Hadoop、Spark、Kafka、Flink、Storm，还有一些 正在孵化的项目，Spring 开源在框架支持上目前已支持的项目有 spring-kafka 和 spring-hadoop。但是 Spring for Hadoop 已有 18个月未更新，当前最新版本是 2.5.0。但是会不会有更多的开源项目加入进来呢？

边缘计算和 IoT，Spring 目前并不适合 IoT 设备，2019 年会不会得到改善？另外，来自 Grails 团队今年开源的新项目 Micronaut，在这方面就比较有优势，目前已经发布 1.0.2 版本，在 2019 年，两个团队会不会再次联手合作呢，我非常期待。

Spring 技术社区，在 2019 年 又将如何做？

  • 翻译官方博客文章，引进新技术、报道新资讯；

  • 深度参与开源项目，提交 issue，PR；

  • 原创系列，技术选型、解决方案、深度解析、开发指南等；

  • 社区活动，举办技术沙龙，分享技术、学习交流、技术培训等;

  • 技术书评，分享最新的技术书籍。

在接下来的一年，Spring 技术社区也会继续发展壮大，欢迎更多的开源技术爱好者参与到社区建设中来，一起学习和交流。

好了，2018 年过去了，我们没有辜负，2019 年来了，期待与你一起去开启新篇章！
