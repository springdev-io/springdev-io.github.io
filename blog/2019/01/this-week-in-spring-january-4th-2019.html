Spring一周回顾（2019年1月4日）

news 
rain
2019-01-04
评论

大家好，Spring 粉丝们，感谢一如既往的等待，又到了每周必读的《Spring一周回顾》时间。

《Spring 2018 年终报告》和《Spring 2018 文章精选合集》，已经正式发布，感谢一年来大家的关注和支持。 可以关注微信公众号： “Spring技术社区”，然后发送关键词 “年终报告” 和 “文章合集” 分别获取相应的电子版（PDF、EPub、Mobi 等格式）。

本周我为大家带来了如下内容：

  • This Year in Spring - December 25th，来自龙大神的年终总结。

  • Spring 技术社区的新年展望，也回顾了过去一年的工作成果。

  • Apache 2018 年度报告已发布，总共 328 个项目，其中顶级项目有 198 个，达到 2 亿行代码。此外，在过去 30 的项目下载数量，中国位居榜首，超过 14 万次。

  • Apache Juneau 8.0.0 已发布

  • Apache Tiles 宣布退休，这也宣告了服务端视图技术逐渐退出舞台，前后端分离已经是大势所趋。

  • Apache HBase 2.0.4 已发布，解决严重问题：在服务器端使用 stream 扫描时内存泄漏、构造 KeyValue 时进行完整性检查

  • Apache ServiceComb Pack 0.3.0 已发布

  • Apache Isis 1.17.0 已发布

  • Apache Drill 1.15.0 已发布，支持索引，自定义 ACL

  • Apache OpenNLP 1.9.1 已发布

  • Apache Groovy 3.0.0-alpha-4 已发布，新增 groovy-yaml 模块，在字节码级别上支持 Java 9/10/11/12

  • Apache OpenMeetings 4.0.7 已发布

  • Apache Qpid Broker-J 7.1.0 已发布

  • Gradle 5.1 已发布，支持依赖库的匹配过滤，支持 Kotlin DSL 1.1.0

  • Serverless 1.35.1 已发布

在知乎上面，我回答了两个初学者的学习 Spring 相关的问题，也许对大家有所帮助：

  • Spring入门该学习什么？

  • 初学Spring有没有适合的书？

历史好文推荐：

  • 上一期的《Spring 一周回顾》同样精彩，值得一读！

  • 《Spring开发指南，带你进入Spring的世界》

  • 《Spring 最全书单，送给爱学习的你》，80本国内外已出版的Spring相关的书，可以根据自己需要选择阅读！

  • 《10 种保护Spring Boot应用程序的绝佳方法》，来自业界安全专家和开源大咖Matt Raible和Simon Maple合作的成果。

  • 《16 条Spring Boot最佳实践》，来自业界Spring大咖们的精彩总结！

周末愉快，感谢阅读，我们下周见！
