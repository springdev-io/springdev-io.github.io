Spring Session Bean-SR2、Apple-SR8 和 1.3.5 同时发布

release 
rain
2019-01-15
评论

Spring Session BOM BEAN-SR2、Apple-SR8 已经同时发布，这两个维护版本分别基于 Spring Session 2.1.3.RELEASE 和 2.0.9.RELEASE，此外，1.3.5.RELEASE 也已发布， 它们解决了一些错误修复以及常规的依赖项升级。

可以在以下更改日志中找到这些版本的完整详细信息：

2.1.3.RELEASE

  • Spring Framework 5.1.4.RELEASE

  • Spring Security 5.1.3.RELEASE

  • Spring Data Lovelace-SR4

  • Reactor Californium-SR4

  • Hazelcast 3.11.1

2.0.9.RELEASE

  • Spring Framework 5.0.12.RELEASE

  • Spring Security 5.0.11.RELEASE

  • Spring Data Kay-SR13

  • Reactor Bismuth-SR15

1.3.5.RELEASE

  • Spring Framework 4.3.19.RELEASE

  • Spring Security 4.2.11.RELEASE
