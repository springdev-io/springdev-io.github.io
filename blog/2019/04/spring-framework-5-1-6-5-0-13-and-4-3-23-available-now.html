Spring Framework 5.1.6、5.0.13和4.3.23 同时发布

release 
rain
2019-04-01
评论

Spring Framework 5.1.6、5.0.13和4.3.23 同时发布，这些版本都是一些问题修复。

Spring Framework 5.1.6 解决了 49 个问题单，5.0.13 作为 5.0.x 版本线最后一次正常发布，也解决了 25 个问题单，在此，也希望所有 5.x 的用户都升级到 5.1.6 版本。与此同时，4.3.23 也解决了 20 个问题单。

与往常一样，以上这些维护版本将会随着下周发布的 Spring Boot 1.5.20、2.0.9、2.1.4 包含进来并可以使用。
