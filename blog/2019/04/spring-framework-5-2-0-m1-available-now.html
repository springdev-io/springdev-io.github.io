Spring Framework 5.2.0.M1 发布

release 
rain
2019-04-10
评论

Spring Framework 5.2.0.M1 已经发布，共解决了 140 项问题修复、改进和依赖项更新，您可以从 Spring Milestone 下载获取。

作为首个里程碑版本，包括以下新特性和缺陷修复：

  • 大量核心容器相关的功能改进，采用新的 MergedAnnotations 接口来解析注解数据，@Configuration 配置类相关的优化

  • 支持 Kotlin 协程

  • spring-webmvc 中增加了新的函数式编程方式 WebMvc.fn，作为 Servlet API 中提供基于使用注解控制器方式的另外一种选择。现在 spring-webmvc 和 spring-webflux 也都提供函数式和基于注解的两种编程方式。

  • Spring MVC 和 Spring WebFlux 的性能改进，减少了请求映射、媒体类型解析、CORS 检查等方面的开销

  • RSocket 支持包括通过带注解的 @MessageMapping 方法进行 响应处理以及通过 RSocketRequester 处理请求

  • 许多有趣的集成测试改进，特别是在测试中处理 应用程序和上下文事件

  • 参考文档重新设计了全新样式

请关注 Spring Framework 5.x 中的新功能和Spring Framework 升级到 5.x 维基页面，因为我们将在里程碑阶段不断添加信息。如果您对当前的版本支持方面感到疑惑，这个 专用页面应该会对您有所帮助。

如果您想在 Spring Boot 应用程序中尝试新的 Spring Framework 功能，那么等 Spring Boot 2.2.0.M2 出来，您就可以使用了。当然，这将会在 https://start.spring.io 上提供。
