Spring一周回顾（2019年3月8日）

news 
rain
2019-03-08
评论

大家好，Spring 粉丝们，感谢一如既往的等待，又到了每周必读的《Spring一周回顾》时间。

本周我为大家带来了如下内容：

Spring 开源社区

  • Spring Boot 2.2 M1 发布，性能持续优化，大量第三方依赖库升级。

  • Spring Cloud Greenwich.SR1 发布，各模块都修复了已知的缺陷和改进。

  • Spring Data Moore M2 发布，带来不少新特性。

  • Spring Cloud Data Flow and Skipper 2.0 GA 发布

  • Spring HATEOAS 1.0 M1 发布，升级到 Java 8 和 Spring Framework 5.1

  • Spring Initializr 新界面改版，界面更加简洁，支持的 Spring Boot 版本一目了然，依赖项由选择框改为搜索下拉框。

安全

  • Keycloak 5.0.0 已发布，升级 WildFly 15

网络-服务器

  • Node 10.15.3、Node 11.11.0 已发布

虚拟机

  • GraalVM 的 Community Edition 1.0 RC13 发布

构建工具

  • Gradle 5.3.0 RC1 已发布

前端

  • Angular 7.2.8 已发布

编辑器

  • Visual Studio Code 1.32.1 已发布

  • Atom v1.35.0 已发布

历史好文推荐：

  • 上一期的《Spring 一周回顾》同样精彩，值得一读！

  • Spring 技术社区的新年展望，《Spring in 2018 年终报告》和《Spring in 2018 文章精选合集》电子版免费下载！

周末愉快，感谢阅读，我们下周见！
