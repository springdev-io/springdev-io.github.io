Spring Boot 2.2 M1 发布

release 
rain
2019-03-08
评论

Spring Boot 2.2 已经发布，现在可以从里程碑版本库中获取。该版本工解决了超过 140 个问题和 PR 请求。

第一次里程碑版本的主要亮点有：

  • Spring Data Moore M2

  • 大量配置属性的绑定速度明显加快

  • 对 bean 初始化时懒加载的可选支持（在随后的博客文章中会有更多内容）

  • JMX 默认不启用

  • 大量第三方依赖库升级

  • 使用 Actuator 时，启动速度更快，内存占用更少

有关更改和升级说明的完整列表，请参阅 Wiki 上的 Spring Boot 2.2 发行说明和升级的参考文档。

如果您想开始使用 2.2 并尝试新功能，可以在 https://start.spring.io 上创建一个新项目。
