Spring一周回顾（2019年3月15日）

news 
rain
2019-03-15
评论

大家好，Spring 粉丝们，感谢一如既往的等待，又到了每周必读的《Spring一周回顾》时间。

本周我为大家带来了如下内容：

Spring 开源社区

  • Andy Wilkinson 分享的精彩博文：JVM 的内存占用量

  • Matt Raible 和 James Ward 在 Devnexus 2019 上的演讲视频

  • Spring Boot 2.2 M1 发布，性能持续优化，大量第三方依赖库升级。

  • Spring Cloud Greenwich.SR1 发布，各模块都修复了已知的缺陷和改进。

  • Spring Data Moore M2 发布，带来不少新特性。

  • SOFABoot 3.1.3 已发布

网络服务器

  • Node 10.15.3、Node 11.11.0 已发布

  • Kubernetes 1.13.4 已发布、Kubernetes 1.12.6 已发布、Kubernetes 1.11.8 已发布

  • Nacos 1.0.0 RC1 已发布

  • Sentinel 1.5.0 已发布

  • Serverless 1.39.0 已发布

  • Kong 1.1.0 rc2 已发布

  • Kibana 6.6.2、Elasticsearch 6.6.2、Logstash 6.6.2 已发布

工作流

  • Activiti 7.1.9 已发布

构建工具

  • Gradle 5.3.0 RC3 已发布

语言

  • Kotlin 1.3.30 EAP 2 已发布

前端

  • Angular 7.2.9 已发布

编辑器

  • Visual Studio Code 1.32.3 已发布

  • Atom v1.35.1 已发布

历史好文推荐：

  • 上一期的《Spring 一周回顾》同样精彩，值得一读！

  • Spring 技术社区的新年展望，《Spring in 2018 年终报告》和《Spring in 2018 文章精选合集》电子版免费下载！

周末愉快，感谢阅读，我们下周见！
