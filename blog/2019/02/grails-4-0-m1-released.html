Grails 4.0 M1 发布

release 
rain
2019-02-19
评论

来自 Object Computing 的 Grails 开发团队最近宣布了 Grails 4.0 M1 已经发布。

Grails 4 作为下一个重要的升级版本，现在框架基于最新的第三方依赖项目重新构建，包括有：

  • Spring 5.1

  • Spring Boot 2.1

  • Java 8

  • Hibernate 5.4

  • Groovy 2.5

此外，Micronaut 现在作为 Grails 的一部分并且作为父 ApplicationContext 包含在内，允许在 Grails 4 应用程序中使用任何 Micronaut 功能，无论是声明性 HTTP 客户端还是 Kafka 客户端。

由于 Spring Boot 2.1 的升级以及 Grails 的一些内部基础组件已经迁移到 Micronaut，所以启动时间和内存消耗也比以前的版本有了显着优化和改进。

许多 Grails 插件已经过验证可以与 Grails 4 一起使用，对于 Grails 插件社区中的人来说，现在是时候升级插件并验证插件是否继续与 Grails 4 一起使用。由于这是一个新的主要版本，API 更改 已经完成，以及 Spring Boot，Groovy 和 Hibernate 中可能需要更改才能与 Grails 4 一起使用。请阅读 升级文档以获取更多信息。
