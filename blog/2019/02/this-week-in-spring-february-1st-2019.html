Spring一周回顾（2019年2月1日，新春快乐！）

news 
rain
2019-02-01
评论

大家好，Spring 粉丝们，感谢一如既往的等待，又到了每周必读的《Spring一周回顾》时间。

此时，正值中国传统佳节——春节，也在这里祝大家新春快乐，万事如意，阖家欢乐，团团圆圆，生活幸福美满，工作步步高升！

本周我为大家带来了如下内容：

  • Spring Cloud Greenwich.RELEASE 发布

  • Spring Tools 4.1.1 发布

  • JHipster v5.8.0 发布

  • Micronaut 1.0.4 发布，模块化和升级第三方依赖。

  • Play Framework 2.7.0 发布，支持 gRPC，使用 Caffeine 作为默认缓存实现，第三方依赖升级。

  • Angular 7.2.3 已发布

  • Kubernetes 1.13.3 已发布

  • Node 10.15.1、Node 11.9.0 已发布

  • Hibernate Search 6.0.0.Alpha2 和 5.11.1.Final 已发布

  • Kong 1.0.3 已发布

  • Gradle 5.2.0 已发布

  • Kubernetes 1.13.3 已发布

  • Kibana 6.6.0、Elasticsearch 6.6.0、Logstash 6.6.0 已发布

历史好文推荐：

  • 上一期的《Spring 一周回顾》同样精彩，值得一读！

  • Spring 技术社区的新年展望，《Spring in 2018 年终报告》和《Spring in 2018 文章精选合集》电子版免费下载！

周末愉快，感谢阅读，我们下周见！
