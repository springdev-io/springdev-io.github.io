Spring Data Lovelace SR5 发布

release 
rain
2019-02-13
评论

Spring Data 团队宣布发布 Lovelace SR5 已经发布，这个版本基于最近发布的 Spring Framework 5.1.5 构建。Spring Boot 在接下来的 2.1.3 版本将选取 Lovelace SR5 以便于使用。

该版本主要包含错误修复和一些第三方依赖升级，总共修复了大约 40 个问题。
