Spring Tools 4.1.2 发布

release 
rain
2019-02-22
评论

Spring Tools 4 是您喜爱的编码环境的下一代 Spring 工具。已经完全从头开始重写，为开发基于 Spring 的企业应用程序提供世界级的支持，无论您喜欢 Eclipse、Visual Studio Code 还是 Atom IDE。

Spring Tools 4 for Eclipse、Visual Studio Code 和 Atom 的 4.1.2 版本已经发布了。

此版本的亮点包括有：

  • (Spring Boot) 新功能：bean 织入的实时悬停信息现在支持在本地服务器安装中运行的 war 包启动应用程序

  • (Spring Boot) 新功能：@Value 注解的实时悬停信息（＃177）

  • (Spring Boot) 新功能：XML 配置文件中的 bean 符号现在包含确切的位置信息

  • (Spring Boot) 修复：导航到正在查看的运行在 CF 上的应用程序的资源已正常

  • (Spring Boot) 修复：现在在搜索项目中的符号发生在服务端，以避免在开始输入查询之前在客户端没有出现与项目相关的符号

  • (Spring Boot) 性能：在检查实时悬停过程时进一步降低 CPU 负载的改进（＃140）

  • (Spring Boot) 性能：语言服务器不再为 Maven 项目触发完整源代码和 javadoc 下载

  • (Concourse) 新功能：支持添加文件中的分层符号，现在可以生成漂亮的大纲视图信息

  • (Concourse) 新功能：支持 YAML 锚点、引用、扩展添加（＃58）

  • (Eclipse) 新功能：快速文本搜索可以切换到非模态模式（＃189）

  • (Eclipse) 新功能：快速文本搜索允许针对特定文件类型过滤结果（＃185）

  • (Eclipse) 修复：在 Eclipse 4.11 的早期版本中发现的启动性能问题

  • 各种其他错误修复和改进

要下载 Eclipse 的发行版，以及找到 Visual Studio Code 和 Atom 的应用市场的链接，请访问：

  • Spring Tools 4: https://spring.io/tools/

详细的更新说明查看这里：

  • https://github.com/spring-projects/sts4/wiki/Changelog#2019-02-21-412-release
