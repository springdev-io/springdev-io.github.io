Spring Boot 2.1.3 发布

release 
rain
2019-02-15
评论

Spring Boot 2.1.3 已经发布，共解决了 70 多项问题修复、改进和依赖项更新，建议使用 2.1.x 的用户升级到此版本。您可以从 Spring Release 和 Maven Central 下载获取。

其中第三方依赖库版本升级：

  • Spring AMQP 2.1.4

  • Spring Data Lovelace-SR5

  • Spring Framework 5.1.5

  • Spring Integration 5.1.3

  • Spring Kafka 2.2.4

  • Spring Security 5.1.4

  • Spring Session Bean-SR3

  • Artemis 2.6.40

  • Byte Buddy 1.9.10

  • Commons Pool2 2.6.1

  • Groovy 2.5.6

  • Infinispan 9.4.6.Final

  • Lettuce 5.1.4.RELEASE

  • Liquibase 3.6.3

  • Log4j2 2.11.2

  • Lombok 1.18.6

  • Micrometer 1.1.3

  • Mimepull 1.9.11

  • Mysql 8.0.15

  • Pooled Jms Version 1.0.4

  • Reactor Bom Californium-SR5

  • Rxjava2 2.2.6

  • Tomcat 9.0.16

  • Undertow 2.0.17.Final

如果您仍在使用 Spring Boot 2.0.x，则强烈建议升级到 Spring Boot 2.1.x。 在宣布 Spring Framework 5.0.x 将于 3 月份达到 EOL 之后，Spring Boot 2.0.x 将在计划于 3 月底推出的 2.0.x 版本的最终版本中也同样到达 EOL。 正如 之前所宣布的那样，Spring Boot 1.5.x 将继续维持，直到 8 月达到其 EOL。

如果您有兴趣参与开源贡献，请查看问题库中的 “ideal for contribution”标签。如果您有一般性问题，请在 stackoverflow.com 上使用 spring-boot 标签或在 Gitter上与社区其他人聊天。
