Spring Cloud Finchley.SR3 发布

Spring Cloud Finchley SR3（Service Release）已经发布，可以在 Maven Central 上获取到。您可以查看Finchley 发布说明以获取更多信息。

模块更新说明

具体而言，下面是每个更新模块的详细介绍。

Spring Cloud Config

  • Spring Cloud Config 服务器现在在检索数据之前遵循请求中的 if-modified-since 标头，并在响应中返回 last-modified 标头

  • 缺陷修复

Spring Cloud Stream

  • Elmhurst.SR2 发布说明

Spring Cloud Sleuth

  • 缺陷修复

Spring Cloud Zookeeper

  • 缺陷修复

Spring Cloud AWS

  • 缺陷修复

Spring Cloud Task

  • 依赖升级

  • 缺陷修复

Spring Cloud Commons

  • 缺陷修复

Spring Cloud Openfeign

  • 缺陷修复

Spring Cloud Vault

  • 依赖升级

  • 文档更新

  • 缺陷修复

Spring Cloud Bus

  • 缺陷修复

Spring Cloud Contract

  • 缺陷修复

Spring Cloud Consul

  • spring.cloud.discovery.enabled=false 现在等同于 spring.cloud.consul.discovery.enabled=false

  • 缺陷修复

Spring Cloud Gateway

  • 当 Ribbon 不在类路径上时，提供更好的错误处理

  • HTTP 和 Websocket 请求路由的性能改进

  • 使用修改请求正文过滤器时的性能改进

  • 缺陷修复

Spring Cloud Netflix

  • HystrixConcurrencyStrategy 可通过 HystrixPlugins#getConcurrencyStrategy 设置

  • spring.cloud.discovery.enabled=false 现在等同于 eureka.client.enabled=false

  • 缺陷修复

Spring 开发者们每周必读的《Spring一周回顾》。

大家好，Spring 粉丝们，感谢一如既往的等待，又到了每周必读的《Spring一周回顾》时间。

本周我为大家带来了如下内容：

  • Spring Tools 4.1.2 发布，增加多项新特性以及带来性能改进。

  • Grails 4.0 M1 发布，升级基础的第三方依赖项，并且集成 Micronaut。

  • Micronaut 1.1.0.M1 发布，模块化、升级第三方依赖，以及 众多新特性发布。

  • SOFABoot 3.1.2 已发布，支持扩展点，并行初始化 bean，可自定义组件生命周期的启用/关闭规则。

  • Hibernate ORM 5.3.8.Final 已发布

  • Hibernate Validator 6.0.15.Final 已发布

  • Eclipse Jetty 9.4.15 已发布

  • Sentinel 1.4.2 已发布

  • RxJava 2.2.7 已发布

  • MariaDB 10.3.13 已发布

  • Angular 7.2.6 已发布

  • Webpack v4.29.5 已发布

  • Serverless 1.38.0 已发布

  • Kong 1.1.0rc1 已发布

  • Kibana 6.6.1、Elasticsearch 6.6.1、Logstash 6.6.1 已发布
